PURPOSE
The purpose of this application is to provide a GUI-based scheduling desktop application for a global consulting organization that conducts business in multiple languages and has main offices in different cities and countries.

IDE: IntelliJ IDEA 2021.3.2 (Community)
JDK: Java 17.0.2
JavaFX: openjfx-18
MySQL: mysql-connector-java-8.0.28

HOW TO RUN
A valid username and password is required based on a match contained in the database and is the first step to login to the system after program start. From there the user may view, add, update, and delete customers and appointments.

REPORTS
The program is capable of generating the following reports via a radio toggle: total number of customer appointments by type and month, a schedule for each contact in the organization, and appointments sorted by division.
